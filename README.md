# Retail-Analytics-and-Churn-Automation.
End-to-end Data Analytics project for Retail Sales and Churn Prediction.

 Project Overview
This project focuses on analyzing retail sales data to perform **RFM (Recency, Frequency, Monetary) Analysis**. It helps in segmenting customers based on their buying behavior to improve targeted marketing strategies.

 Tech Stack
- **Language:** Python (Pandas, NumPy)
- **Concepts:** Customer Segmentation, Data Cleaning, RFM Modeling

 Key Insights
- **Recency:** How recently a customer made a purchase.
- **Frequency:** How often they purchase.
- **Monetary:** How much revenue they generate.
- Segmented customers into categories like **'Champions'**, **'Loyal Customers'**, and **'At Risk'**.

 File Structure
- `rfm_analysis.py`: Python script for calculating RFM scores.
- `clean_customer_data.csv`: Pre-processed retail dataset.
- `rfm_output.csv`: Final segmented customer data.
